# Carousel

## Setting Props

```
### auto
#### type: boolean
#### default: true

### height
#### type: number
#### default 400

### speed
#### type: number
#### default: 3000

```

## Example

```
<Carousel {...settings}>
  <img src="images/test_img1.jpg" alt="img" />
  <img src="images/test_img2.jpg" alt="img" />
  <img src="images/test_img3.jpg" alt="img" />
  <img src="images/test_img4.jpg" alt="img" />
</Carousel>
```
